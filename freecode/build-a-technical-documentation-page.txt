    <section class="main-section" id="Variables">
      <header>Variables</header>
      <p>Variables are containers for storing data values.</p>
      <p>In JavaScript, you can declare variables using var, let, or const.</p>
      <code>let x = 10;</code>
      <code>const y = "Hello";</code>
      <ul>
        <li>var - function scoped</li>
        <li>let - block scoped</li>
        <li>const - block scoped, cannot be reassigned</li>
      </ul>
    </section>

    <section class="main-section" id="Functions">
      <header>Functions</header>
      <p>Functions are blocks of reusable code.</p>
      <p>You can call a function to execute code multiple times.</p>
      <code>function greet(){ return "Hello"; }</code>
      <p>Functions can take parameters and return values.</p>
      <ul>
        <li>Reusable</li>
        <li>Maintainable</li>
      </ul>
    </section>

    <section class="main-section" id="Loops">
      <header>Loops</header>
      <p>Loops are used to execute a block of code multiple times.</p>
      <p>Different types of loops are available in JavaScript.</p>
      <code>for(let i=0; i<5; i++){ console.log(i); }</code>
      <ul>
        <li>for loop</li>
        <li>while loop</li>
        <li>do...while loop</li>
      </ul>
    </section>

    <section class="main-section" id="Objects">
      <header>Objects</header>
      <p>Objects are collections of key-value pairs.</p>
      <p>They allow storing structured data.</p>
      <code>let car = {brand:"Toyota", model:"Corolla"};</code>
      <p>You can access object properties using dot or bracket notation.</p>
      <ul>
        <li>Properties</li>
        <li>Methods</li>
      </ul>
    </section>
